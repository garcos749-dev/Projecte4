# **T08: Auditoria de Qualitat i Estandardització de Servidors**

## **Versió estil “punt 8”**

## **1. Introducció**

1. En les darreres setmanes s’han implementat serveis complexos:

   * NFS (servidors de fitxers)
   * CUPS (impressió)
   * LDAP (autenticació)
2. Cap servei avançat pot funcionar correctament si el sistema operatiu no és sòlid.
3. Un servidor amb problemes de base (xarxa mal configurada, permisos incorrectes o paquets obsolets) provoca:

   * Riscos de seguretat
   * Incidències futures
4. A EverPia, la qualitat i robustesa són part de la reputació del servei.
5. Aquesta tasca posa el focus en els **fonaments**: abans de desplegar res avançat, s’ha de garantir un servidor base impecable.

---

# **2. La vostra missió**

1. Fer una **auditoria interna** i aplicar el **Protocol de Desplegament Estàndard (SOP)**.
2. Demostrar que podeu desplegar una màquina nova (Ubuntu Server) i configurar-la correctament.
3. Validar que domineu les tasques bàsiques essencials abans de treballar a un entorn de client.
4. Aquest entrenament forma part del procés per obtenir la certificació **SOP-Specialist**.

---

# **3. Fase d’Entrenament**

1. Utilitzareu una OVA proporcionada pel docent.
2. Haureu de completar totes les tasques indicades al document **Projecte4-TascaSOP** (descarregable).
3. No cal lliurar cap activitat en aquesta fase.
4. És essencial comprendre perfectament totes les tasques:

   * Configuració bàsica del sistema
   * Xarxa i hostname
   * Actualitzacions i repositoris
   * Gestió d’usuaris i permisos
   * Configuració de firewall
   * Serveis bàsics i verificació
5. És una fase d'aprenentatge i consolidació.

---

# **4. Certificació SOP-Specialist**

1. Un cop completada la fase d’entrenament, fareu una **prova pràctica d’alt nivell**.
2. L’examen validarà:

   * Configuració correcta del servidor
   * Execució del protocol SOP
   * Capacitat de resolució i ordre metodològic
3. Normes de l’examen:

   * Només podeu portar **un únic full manuscrit** amb anotacions pròpies.
   * No es permeten apunts impresos ni materials addicionals.
4. Superar la prova = obtenir la certificació **SOP-Specialist**.
